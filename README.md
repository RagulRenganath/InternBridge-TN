InternBridge TN is a high-fidelity, single-page application (SPA) designed to solve structural employment challenges by connecting Tier-2 and Tier-3 college students in Tamil Nadu with thriving SMEs and Corporate partners. 

Built with a futuristic "Bento-grid" UI, the platform features a custom JavaScript state-management engine that routes seamlessly between three distinct user portals without page reloads.

## ✨ Key Features

### 🎓 1. Student Portal (Growth & Prep)
* **Dynamic Opportunity Board:** Filterable listings for Finance, Economics, Tech, and Marketing roles.
* **Prep & Motivation Hub:** High-quality, curated YouTube video feeds for aptitude, group discussions, and interview mindset.
* **Industry News Engine:** Real-time localized market updates for the Tamil Nadu business landscape.
* **Pipeline Tracking:** Visual Kanban-style tracker for active internship applications.

### 💼 2. Recruiter Hub (SME Partners)
* **Hiring Analytics:** Real-time data visualization for Time-to-Hire, Active Pipeline, and Offer Acceptance Rates.
* **Candidate Discovery:** Searchable database of verified students with automated "AI Match Scores".
* **Drag-and-Drop Pipeline:** Interactive stage management (Applied ➔ Interview ➔ Offer ➔ Rejected).

### 🛡️ 3. Governance Deck (Placement Admins)
* **Cohort Intelligence:** Macro-level data tracking application volume and placement success across departments (MBA, B.Tech, BA Economics).
* **Approval Queue:** Quality-control interface to review and approve SME internship posts before they go live.

## 🛠️ Tech Stack & Architecture
* **Frontend:** HTML5, Vanilla JavaScript (ES6+).
* **Styling:** Tailwind CSS (via CDN) with custom glassmorphism and modern gradient utility classes.
* **Icons:** Lucide Icons.
* **Architecture:** Component-based rendering pattern contained entirely within a single `index.html` file, utilizing a centralized `AppState` object for instantaneous DOM mutations.
